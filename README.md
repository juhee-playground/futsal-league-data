## 📊 순위표(Standings) 업데이트 방법

본 프로젝트는 `matches.json`의 경기 결과를 바탕으로 `standings.json`을 자동 계산하는 스크립트를 포함하고 있습니다. 경기 결과가 새로 입력되거나 수정되었을 때, 아래 방법으로 순위표를 수동 업데이트할 수 있습니다.

### 1. 실행 전 확인 사항

- Node.js가 설치되어 있어야 합니다. (`node -v`로 확인)
- 데이터 파일 경로가 다음과 같은지 확인하세요:
  - `2025-2026/division/fk1/`
  - `2025-2026/division/fk2/`
  - `2025-2026/division/wk/`

### 2. 실행 방법

터미널에서 아래 명령어를 실행하면 모든 리그의 **순위표(Standings)**와 **득점 순위(Scorers)**가 동시에 갱신됩니다.

```bash
node update_all.js
```

### 💡 참고사항

이제 `matches.json`에 경기가 끝날 때마다 누가 골을 넣었는지 `goals` 배열에 차곡차곡 쌓아주기만 하면 됩니다. 스크립트가 알아서 모든 경기를 뒤져서 선수별 총 골 수를 계산하고 정렬까지 해줄 거예요!
